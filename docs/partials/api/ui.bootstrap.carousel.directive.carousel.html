carousel
directive in module ui.bootstrap.carousel

Description

Carousel is the outer container for a set of image 'slides' to showcase.

Usage

as element:
<carousel
       [interval="{number}"]
       [no-transition="{boolean}"]
       [no-pause="{boolean}"]>
</carousel>
as attribute
<ANY carousel
     [interval="{number}"]
     [no-transition="{boolean}"]
     [no-pause="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
interval
(optional)
number

The time, in milliseconds, that it will take the carousel to go to the next slide.

noTransition
(optional)
boolean

Whether to disable transitions on the carousel.

noPause
(optional)
boolean

Whether to disable pausing on the carousel (by default, the carousel interval pauses on hover).
