Data
service in module main

Description

This is the main service for retrieving data at regular intervals.

Dependencies

  • $rootscope
  • $http
  • net
  • cvt

Methods

  • handlePAS(d, Data, shift)

    This function handles allocation of the PAS data. All data may be plotted and as such the data is divided up into arrays of {x,y} pairs for use by plotting libraries. The length of the arrays is defined by the service and the length is indicated by the input shift.

    Parameters
    ParamTypeDetails
    dObject

    The JSON data object returned by the server.

    DataObject

    Data object that will be broadcasted to controllers.

    shiftboolean

    Indicates whether we have the correct number of points in the array and need to start shifting the data.

    Returns
    Object

    Data object defined in the inputs.

  • updateTime(t)

    Takes the time returned by the server (a LabVIEW time) and converts it to a Javascript Date.

    Parameters
    ParamTypeDetails
    tnumber

    Time in seconds since January 1, 1904.

    Returns
    Date

    Date object with date from server.

Properties

  • alicats

    Defines a list of alicat names to ID data related to a specific alicat device.

  • maxLength

    Defines the max array length in seconds for displaying data.

  • ppts

    Defines a list of ppt names to ID data related to a specific ppt device.

  • shiftData

    Determines how to shuffle array data (used in conjunction with maxLength). If the value is true, the number of elements in the arrays is >= maxLength.

  • vaisalas

    Defines a list of vaisala names to ID data related to a specific vaisala device.
