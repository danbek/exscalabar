net
service in module main

Description

This service handles network settings that can be set on the configuration page. This communicates the settings to the other portions of the application that require the ip address and port.

This service stores the settings in local storage so that they are restored on refresh.

Methods

  • getNetworkParams()

    Returns
    object

    Object containing two keys: ip and port. These values are strings containing the IP address and the port that the server is listening on.

  • setNetworkParams(ip, port)

    Sets the network settings for communicating with the server. These values are cached using html5 localstorage.

    Parameters
    ParamTypeDetails
    ipstring

    IP address of server

    portstring

    Port server is listening on.
