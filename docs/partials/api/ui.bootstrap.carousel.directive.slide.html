slide
directive in module ui.bootstrap.carousel

Description

Creates a slide inside a carousel. Must be placed as a child of a carousel element.

Usage

as element:
<slide
       [active="{boolean}"]>
</slide>
as attribute
<ANY slide
     [active="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
active
(optional)
boolean

Model binding, whether or not this slide is currently active.
